class DAC_ODEsolverRKexplicit : public DAC_ODEsolver¶

Class implementing the abstract function step for the advancing using explicit Runge-Kutta methods. The user must simply define the Tableau of the Runge-Kutta method

\[\begin{split}\begin{array}{c|c} c & A \\ \hline & b^T \end{array}\end{split}\]

Subclassed by Collatz, ExplicitEuler, Heun, Heun3, Midpoint, RK3, RK3_8, RK4, Ralston, Ralston3, Ralston4, SSPRK3
