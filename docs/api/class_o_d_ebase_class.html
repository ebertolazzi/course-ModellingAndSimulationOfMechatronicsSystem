class ODEbaseClass : public handle¶

ODE base class describing an ODE

Usage 

 ode = ODE_derived_class() % create an instance of a ODE
 res = ode.f(t,x);

Subclassed by DAE3baseClassImplicit

Public Functions

virtual f(t,x)¶

Abstract functions. The derived function must define the ODE \( x' = f( t, x ) \)

virtual DfDx(t,x)¶

Abstract functions. The derived function must return the jacobian \( \partial f( t, x ) / \partial x \)

virtual DfDt(t,x)¶

Abstract functions. The derived function must return the jacobian \( \partial f( t, x ) / \partial t \)

virtual exact(t0,x0,t)¶

Abstract functions. The derived function must return the exact solution \( x(t) \) of the ODE \( x' = f( t, x ) \) such that \( x(t_0) = x_0 \). If exact solution is not available the derived function that return []
