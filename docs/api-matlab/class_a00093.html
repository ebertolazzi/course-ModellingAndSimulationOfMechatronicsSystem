class DAC_ODEclass : public DAC_base_class¶

ODE base class describing an ODE

Usage

ode = ODE_derived_class() % create an instance of a ODE
res = ode.f(t,x);

Public Functions

virtual f(t,x)¶

Abstract functions defining and ODE

\[ \mathbf{x}' = \mathbf{f}( t, \mathbf{x} ) \quad\oplus\quad [\textrm{invariants}] \]

invariants are of the form

\[ \mathbf{h}( t, \mathbf{x} ) = \mathbf{0} \]

The derived function must define the r.h.s. of the ODE \( \mathbf{f}( t, \mathbf{x} ) \) where \( \mathbf{x} \) are the states of the model described by the ODE.

virtual DfDx(t,x)¶

The derived function must return the jacobian of the r.h.s. of the ODE:

\[ \partial \mathbf{f}( t, \mathbf{x} ) / \partial \mathbf{x} \]

Used only in implicit methods

virtual h(t,x)¶

The invariant/hidden constraints \( \mathbf{h}( t, \mathbf{x} ) = \mathbf{0} \)

virtual DhDx(t,x)¶

The derived function must return the jacobian of the hidden constraints of the ODE:

\[ \partial \mathbf{h}( t, \mathbf{x} ) / \partial \mathbf{x} \]

Used only in projection method
