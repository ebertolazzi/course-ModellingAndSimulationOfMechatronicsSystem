virtual function f( this, t, x, ignoredArg)¶

(Abstract) Definition of the system of ODEs:

\[ \mathbf{x}' = \mathbf{f}( t, \mathbf{x} ) \quad\oplus\quad [\textrm{invariants}] \]

invariants are of the form:

\[ \mathbf{h}( t, \mathbf{x} ) = \mathbf{0} \]

The derived function must define the r.h.s. of the ODE \( \mathbf{f}( t, \mathbf{x} ) \) where \( \mathbf{x} \) are the states of the model described by the ODE.

virtual function DfDx( this, t, x, ignoredArg)¶

(Abstract) Evaluate the jacobian of the r.h.s. of the system of ODEs:

\[ \partial \mathbf{f}( t, \mathbf{x} ) / \partial \mathbf{x} \]

Used only in implicit Runge-Kutta methods

virtual function h( this, t, x, ignoredArg)¶

(Abstract) Evaluate the invariants/hidden constraints of the system of ODEs:

\[ \mathbf{h}( t, \mathbf{x} ) = \mathbf{0} \]
virtual function DhDx( this, t, x, ignoredArg)¶

(Abstract) Evaluate the Jacobian of the hidden constraints of the system of ODEs:

\[ \partial \mathbf{h}( t, \mathbf{x} ) / \partial \mathbf{x} \]

Used only in projection method
